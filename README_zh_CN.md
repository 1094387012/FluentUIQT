## 条件

+ Vs2022, Qt6.5.1

## 开始

+ 克隆仓库.

```SHELL
git clone https://github.com/mowangshuying/CppQtFluentUi888.git
```

+ Build

>1.用Vs2022打开CppQt.
>
>2.编译!
>
>3.设置为启动项目并且启动!
